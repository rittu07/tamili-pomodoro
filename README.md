# Tamil Pomodoro Timer

Tamil Pomodoro Timer is a unique productivity tool that combines the Pomodoro technique with a touch of Tamil humor. It helps you stay focused with timed work intervals, funny break reminders, and distraction blocking.

## Features

-   **Pomodoro Timer**: Integrated Pomodoro timer (30-minute work intervals) to keep you focused.
-   **Break Reminders**: Light-hearted Tamil phrases and alerts to remind you to stretch and relax.
-   **Distraction Blocking**: Aggressively blocks social media sites during work sessions.
-   **Usage Tracking**: Monitors your screen time and web usage statistics.
-   **YouTube Shorts Blocker**: Prevents endless scrolling on YouTube Shorts.

## Technologies Used

-   **Frontend**: HTML, CSS, JavaScript
-   **Extension**: Chrome Extension Manifest V3
-   **APIs**: `chrome.tabs`, `chrome.alarms`, `chrome.notifications`, `chrome.webNavigation`

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/rittu07/tamil-pomodoro-timer.git
    ```
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode**.
4.  Click **Load unpacked** and select the `tamil-pomodoro-timer` directory.

## Usage

-   Click the extension icon to view your dashboard and timer status.
-   The extension automatically tracks site usage and will alert you when it's time for a break or if you're visiting a blocked site.
